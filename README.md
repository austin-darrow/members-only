# README

This project features an exclusive web app where members can write anonymous posts. Members can see who the author of a post is, but non-members cannot.

Features an authentication system so users can only access areas of the site they are authorized to.

This project is part of the Rails track from The Odin Project. The full assignment can be found here: https://www.theodinproject.com/lessons/ruby-on-rails-members-only
